Política de Privacidade do Purrinha Game

Última atualização: 22 de setembro de 2024

1. Introdução

Bem-vindo ao Purrinha Game, desenvolvido por VitorinoPeixoto. Este documento esclarece como lidamos com quaisquer informações que você nos fornece enquanto joga e como interagimos com terceiros para exibir publicidade.

2. Informações que Coletamos

a) Informações inseridas pelo jogador:
Nosso jogo solicita que você insira seu nome no início para personalizar a experiência de jogo. Este nome é usado exclusivamente durante a sessão de jogo e não é armazenado em nossos servidores, nem compartilhado diretamente com terceiros.

b) Dados de uso e técnico:
Para fornecer uma melhor experiência e exibir anúncios relevantes, podemos coletar automaticamente algumas informações sobre seu dispositivo, como o tipo de dispositivo, sistema operacional, identificador de publicidade e dados de uso do jogo. Estes dados podem ser coletados por serviços de terceiros, como provedores de anúncios.

c) Publicidade:
Nosso jogo exibe anúncios fornecidos por parceiros publicitários, que podem utilizar cookies e tecnologias similares para coletar dados com a finalidade de mostrar anúncios personalizados. Estes dados incluem identificadores de dispositivo, localização aproximada e comportamento de uso. O controle sobre esses dados depende das configurações do seu dispositivo e dos provedores de anúncios.

3. Uso de Informações

a) Dados de sessão:
O nome que você insere é utilizado apenas durante a sessão de jogo para personalização e interação no jogo. Não armazenamos esse nome após o término da sessão.

b) Dados para anúncios:
Os dados coletados pelos provedores de anúncios são usados para exibir anúncios personalizados e melhorar a relevância dos mesmos. Esses dados podem ser usados para fins de análise de desempenho e melhoria da experiência de publicidade.

4. Compartilhamento de Informações

a) Parceiros de publicidade:
Podemos compartilhar informações não identificáveis pessoalmente, como dados de uso e identificadores de publicidade, com terceiros (como redes de anúncios) para exibir publicidade em nosso jogo. Esses dados são coletados e processados de acordo com as políticas de privacidade desses provedores de publicidade.

b) Informações do usuário:
Fora o compartilhamento necessário com parceiros publicitários, não compartilhamos informações pessoais ou identificáveis com terceiros.

5. Segurança

A segurança de suas informações é importante para nós. Implementamos medidas para proteger as informações coletadas contra acesso não autorizado ou alteração. No entanto, como lidamos com provedores de terceiros, não podemos garantir a segurança total das informações processadas por essas plataformas.

6. Seus Direitos

Você pode ajustar as configurações de privacidade e publicidade no seu dispositivo Android para limitar a coleta de dados para publicidade personalizada. Consulte as configurações de privacidade do dispositivo para desativar anúncios personalizados ou limitar o acesso a dados de localização.

7. Alterações nesta Política

Podemos atualizar nossa política de privacidade periodicamente para refletir mudanças nas nossas práticas operacionais ou de publicidade. Quaisquer alterações serão postadas nesta página com uma nova data de "última atualização".
